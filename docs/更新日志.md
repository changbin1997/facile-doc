##Facile 2.3

更新日期：2026年1月19日，更新内容：

1. 语言: 加入了多语言支持，目前支持简体中文和英文
2. 主题设置: 后台的主题设置的友情链接设置加入了链接编辑器
3. 图片灯箱: 重写了图片灯箱，加入了切换图片和可访问性优化
4. 导航栏: 导航栏的自定义链接加入了下拉菜单支持
5. 目录: 侧边栏章节目录加入了目录高亮显示，可根据当前浏览的区域高亮对应的目录列表项
6. 目录: 调整了章节目录链接的生成方式，从之前的随机值改为按顺序生成固定地址
7. 搜索页: 加入了 noindex 标签设置和显示，可以有效避免网站因被垃圾广告机器人频繁搜索而在 Google 等搜索结果中出现大量无效广告页面
8. 文章页: 加入了版权信息显示和设置
9. BUG修复: 修复了代码块行号显示错误的问题
10. BUG修复: 修复了在没有任何文章和页面的情况下，数据库初始化错误的问题
11. BUG修复: 修复了统计页英文日期出错的问题
12. BUG修复: 修复了第一次使用时，MySQL 可能会初始化错误的问题
13. 优化: 评论区的 Emoji 面板使用的表情从之前的请求 PHP 改为直接内置到 JS
14. 优化: 评论表单的必填表单加入了 `*` 提示
15. 优化: 调整了在没有任何文章和分类的情况下，统计页图表的显示方式
16. 优化: 文章编辑链接的 admin 管理页面地址从之前的固定地址改为使用 API 获取
17. 优化: 其它可访问性优化
